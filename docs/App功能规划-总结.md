# 灵值生态园 App - 功能规划总结

**版本**: v1.0
**创建日期**: 2025年1月15日
**主体公司**: 陕西媄月商业艺术有限责任公司

---

## 一、项目概述

灵值生态园 App 是一个基于AI智能体的情绪价值平台，核心功能是为用户提供情绪识别、情感支持、成长建议等服务。本规划文档详细描述了App需要实现的功能、页面和交互设计。

---

## 二、核心功能模块（8个）

### 1. 用户系统模块
- 用户注册/登录
- 自动注册登录（一键登录）
- 用户信息管理
- 实名认证
- 收款方式管理
- 密码管理
- 登录状态管理

### 2. 签到系统模块
- 每日签到
- 签到历史查询
- 签到状态查询
- 签到统计
- 灵值奖励发放

### 3. 情绪系统模块（核心）
- 情绪识别（6种基础情绪：开心、悲伤、愤怒、焦虑、惊讶、平静）
- 情绪记录（保存到数据库）
- 情绪统计（日/周/月）
- 情绪日记（创建、编辑、删除、查看）
- 情绪模式分析
- 情绪趋势图表

### 4. 智能体系统模块
- 智能对话（与灵值智能心灵伙伴对话）
- 情绪识别与共情回应
- 成长建议
- 对话历史管理
- 对话设置

### 5. 公司动态模块
- 发布公司动态（管理员）
- 查看公司动态列表
- 查看动态详情

### 6. 财务系统模块
- 灵值查看
- 灵值明细查询
- 灵值兑换
- 提现申请
- 提现审核（管理员）

### 7. 个人中心模块
- 个人信息展示
- 个人信息编辑
- 头像上传
- 签到记录查询
- 灵值记录查询

### 8. 设置模块
- 账号设置
- 修改密码
- 双因素认证
- 隐私设置
- 对话设置
- 关于我们
- 帮助与反馈
- 退出登录

## 四、核心页面功能详解

### 1. 首页（Home）

**功能列表**：
- ✅ 每日签到（显示签到状态、连续签到天数、灵值奖励）
- ✅ 公司动态（显示最新3条动态）
- ✅ 快捷入口（快速进入对话页、情绪记录页）

**交互流程**：
1. 打开App → 自动检查签到状态
2. 点击"签到"按钮 → 签到成功动画
3. 点击公司动态 → 查看动态详情
4. 点击快捷入口 → 跳转到对应页面

---

### 2. 对话主页（ChatHome）

**功能列表**：
- ✅ 智能对话（与灵值智能心灵伙伴对话）
- ✅ 实时情绪识别（自动识别用户情绪）
- ✅ 个性化回应（基于情绪的智能回应）
- ✅ 快捷情绪选择（快速选择当前情绪）
- ✅ 对话历史（显示历史对话记录）

**交互流程**：
1. 打开对话页 → 显示欢迎消息
2. 输入消息 → 发送给AI
3. AI回复 → 显示回复内容
4. 自动识别情绪 → 显示情绪标签

---

### 3. 情绪主页（EmotionHome）

**功能列表**：
- ✅ 今日情绪状态（显示今日主要情绪、强度）
- ✅ 本周情绪趋势（显示7天情绪变化图表）
- ✅ 我的情绪日记（显示最近3条日记）
- ✅ 情绪成长建议（基于情绪模式的个性化建议）

**交互流程**：
1. 打开情绪主页 → 显示今日情绪状态
2. 点击情绪卡片 → 跳转到情绪记录页
3. 点击情绪趋势 → 跳转到情绪统计页
4. 点击日记 → 查看日记详情

---

### 4. 情绪记录页（EmotionRecord）

**功能列表**：
- ✅ 选择情绪类型（6种基础情绪选择）
- ✅ 设置情绪强度（滑动条调节）
- ✅ 添加情绪上下文（可选的文本输入）
- ✅ 保存情绪记录

**交互流程**：
1. 选择情绪 → 高亮显示
2. 调节强度 → 实时显示强度值
3. 输入上下文（可选）
4. 点击保存 → 保存成功提示

---

### 5. 情绪统计页（EmotionStatistics）

**功能列表**：
- ✅ 时间范围选择（日/周/月切换）
- ✅ 情绪分布图表（饼图或柱状图）
- ✅ 情绪统计列表（显示每种情绪的次数和百分比）
- ✅ 导出报告（导出统计报告）

**交互流程**：
1. 切换时间范围 → 图表自动刷新
2. 查看图表 → 显示情绪分布
3. 查看统计列表 → 显示详细数据
4. 导出报告 → 生成并分享报告

---

### 6. 情绪分析页（EmotionAnalysis）

**功能列表**：
- ✅ 情绪模式分析（最常见情绪、情绪稳定性、平均强度）
- ✅ 情绪趋势（显示情绪变化趋势图）
- ✅ 成长建议（基于情绪模式的个性化建议）

**交互流程**：
1. 打开分析页 → 显示情绪模式分析
2. 查看趋势图 → 显示情绪变化趋势
3. 查看建议 → 显示成长建议

---

### 7. 个人中心（Profile）

**功能列表**：
- ✅ 用户信息展示（头像、姓名、邮箱）
- ✅ 灵值余额（显示灵值余额）
- ✅ 个人信息管理（手机号、邮箱、实名认证、收款方式）
- ✅ 签到记录（显示连续签到天数、总签到次数）

**交互流程**：
1. 打开个人中心 → 显示用户信息
2. 点击灵值余额 → 查看灵值明细
3. 点击个人信息 → 编辑个人信息
4. 点击签到记录 → 查看签到详情

---

### 8. 设置主页（Settings）

**功能列表**：
- ✅ 账号与安全（账号设置、修改密码、双因素认证）
- ✅ 隐私设置（对话隐私、数据管理、权限管理）
- ✅ 对话设置（对话主题、消息通知、语音播报）
- ✅ 关于（关于我们、用户协议、隐私政策、帮助与反馈）
- ✅ 退出登录

**交互流程**：
1. 打开设置页 → 显示设置列表
2. 点击设置项 → 跳转到对应页面
3. 完成设置 → 自动保存
4. 退出登录 → 清除本地数据

## 六、技术架构

### 前端技术栈
- **框架**: React Native 0.72+ 或 Flutter 3.0+
- **状态管理**: Redux Toolkit / Provider
- **UI组件**: React Native Paper / Material Design
- **图表库**: Victory Native / FL Chart
- **导航**: React Navigation 6

### 后端技术栈
- **语言**: Python 3.10+
- **框架**: FastAPI 0.100+
- **数据库**: PostgreSQL 15+
- **ORM**: SQLAlchemy 2.0+

### AI服务
- **智能体**: 灵值智能体v8.0
- **模型**: 豆包（Seed）大模型
