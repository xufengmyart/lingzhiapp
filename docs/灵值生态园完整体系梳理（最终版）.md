- **获取方式**：签到、充值、活动奖励、邀请奖励、任务完成
- **用途**：购买服务、解锁功能、兑换礼品、参与活动

#### 灵值智能体
- **名称**：灵值
- **角色**：智能心灵伙伴
- **核心能力**：情绪识别、共情回应、成长建议、情感支持
- **特点**：有温度、有个性、有记忆、能成长

#### 📺 P009 中视频项目

**项目概述**
中视频项目是灵值生态园的核心内容创作板块，专注于制作1-10分钟的优质中视频内容，传递文化价值与美学理念。

**内容定位**
- **文化美学类**：传统文化、艺术鉴赏、美学理论
- **生活美学类**：生活方式、美学实践、创意灵感
- **城市美学类**：西安本地文化、历史故事、城市探索
- **商业美学类**：品牌故事、营销美学、设计思维

- 格式：MP4、MOV

### 7.2 核心业务流程

#### 流程1：用户注册流程
```
[开始] → 输入手机号 → 获取验证码 → 输入验证码 → 设置密码
→ 输入昵称 → 输入邀请码(可选) → 同意协议 → [提交]
→ 注册成功 → 自动登录 → 进入新手引导 → [结束]
```

#### 流程2：每日签到流程
```
[开始] → 进入签到页 → 查看今日状态 → 点击签到
→ 签到成功动画 → 显示获得灵值 → 查看连续天数
→ 分享签到(可选) → [结束]
```

#### 流程3：充值流程
```
[开始] → 进入充值页 → 选择充值档位 → 查看档位详情
→ 点击充值 → 确认订单 → 选择支付方式 → 发起支付
→ 支付中... → 支付完成 → 灵值到账 → 查看余额 → [结束]
```

#### 流程4：合伙人申请流程
