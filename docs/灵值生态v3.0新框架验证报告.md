### 【核心路由】工作流

#### 基础信息
- **名称**：核心路由 - 用户意图分析与响应引导
- **触发方式**：对话开始时自动触发
- **输出**：
  - `商业效率`：具体、事务性、求结果的问题
  - `文化深度`：抽象、求理解、探索性的问题
  - `混合咨询`：同时涉及两者，或意图模糊

#### 核心节点
1. **开始节点**：接收用户输入
2. **LLM判断节点**：
   - 模型：doubao-seed-1-6-251015
   - 温度：0.3
   - 系统提示词：详细的意图分类规则
3. **分支判断节点**：三个分支（商业效率/文化深度/混合咨询）
4. **变量赋值节点**：设置`intent_type`和`reply_strategy`
5. **输出节点**：返回意图类型和回复策略

#### 集成方式
- **方式1**：每次对话自动调用工作流节点
- **方式2**：通过系统提示词引用`{{intent_type}}`变量
