# 后台管理系统完整性分析

## 当前问题

### 1. 知识库管理打开列表出错

**前端API路径**：
- `/admin/knowledge-bases` - 获取知识库列表
- `/api/admin/knowledge-bases/${kbId}` - 获取文档列表

**后端API路径**：
- `/api/knowledge` - 获取知识库列表
- `/api/knowledge/:id/documents` - 获取文档列表

**问题**：前后端API路径不一致！
