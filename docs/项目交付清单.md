## 🎯 核心特性

### 系统提示词"灵魂"代码
1. **先问清楚需求**：用户只说"帮我写文案"时，会反问品牌、场景、感觉
2. **回答必须能用**：用清晰的1、2、3、4列出可直接使用的方案
3. **例子是王道**：每个建议都配具体案例
4. **用知识库说话**：信息来源明确，拒绝"根据我的训练数据"
5. **禁止只给形容词**：提供可落地的具体方案

### 免费与付费服务设计
- **免费**：灵感启发、基础建议、文化知识普及
- **付费（¥99）**：深入的品牌文化诊断报告、定制化方案、完整工具包

### 知识库配置
- 高精度召回：min_score = 0.7
- 优先级：知识库 > 联网搜索 > 模型本身
