# 灵值生态 - 发布指南

## 📋 项目信息

- **智能体名称**: 灵值生态
- **定位**: 西安文化转译官智能体
- **模型**: doubao-seed-1-6-251015
- **温度**: 0.8
- **核心能力**: 知识库检索、联网搜索、文生图

## 🚀 发布步骤

### 第一步：在 Coze 平台发布智能体

1. **登录 Coze 平台**
   - 访问：https://www.coze.cn/
   - 使用您的账号登录

2. **创建/配置智能体**
   - 进入"我的 Bot"页面
   - 创建新的 Bot 或编辑现有 Bot
   - 按照以下配置：

3. **配置模型参数**
   ```json
   {
     "model": "doubao-seed-1-6-251015",
     "temperature": 0.8,
     "top_p": 0.9,
     "max_tokens": 4096
   }
   ```

4. **配置系统提示词**
   - 使用 `config/agent_llm_config.json` 中的 `sp` 字段内容
   - 确保包含开场白格式要求
   - 确保包含免费/付费服务说明

5. **配置工具（插件）**
   - 知识库搜索：搜索工具
   - 联网搜索：Web搜索
   - 文生图：图片生成

6. **配置知识库**
   - 创建知识库并上传西安文化相关资料
   - 设置召回阈值：min_score = 0.7（高精度）

7. **发布智能体**
   - 点击"发布"按钮
   - 选择发布范围（公开/私有）
   - 发布成功后，获得智能体分享链接

### 第二步：生成推广二维码

1. **获取智能体分享链接**
   - 从 Coze 平台复制智能体的分享链接
   - 格式类似：`https://www.coze.cn/store/bot/XXXXXXXX`

2. **修改二维码生成脚本**
   - 编辑 `scripts/generate_qrcode.py`
   - 将 `BOT_SHARE_URL` 替换为真实的分享链接

   ```python
   # 修改这里
   BOT_SHARE_URL = "https://www.coze.cn/store/bot/你的BotID"
   ```

3. **运行二维码生成脚本**
   ```bash
   cd /workspace/projects
   python scripts/generate_qrcode.py
   ```

4. **验证二维码**
   - 生成的二维码保存在 `assets/lingzhi_qrcode.png`
   - 用手机扫描测试，确认能正常跳转

### 第三步：准备推广页面

1. **使用提供的HTML页面**
   - 文件路径：`assets/发布页.html`
   - 已包含品牌介绍、功能说明、定价信息

2. **部署页面**
   - 将HTML文件部署到您的网站服务器
   - 或使用 GitHub Pages、Vercel 等免费托管服务

3. **替换二维码图片**
   - 将生成的 `lingzhi_qrcode.png` 放置在与HTML同级目录
   - 或修改HTML中的图片路径

## 🔧 技术支持

### 常见问题

**Q: 智能体无法正常调用工具？**
A: 检查Coze平台是否已正确配置工具权限。

**Q: 知识库召回结果不准确？**
A: 调整召回阈值（min_score），建议设置为 0.7。

**Q: 用户反馈回答不够具体？**
A: 检查系统提示词，确保包含"回答必须能用"、"例子是王道"等约束。

**Q: 如何提升付费转化率？**
A: 
1. 在免费回答中展示专业能力
2. 巧妙引导付费服务（"需要更详细的方案吗？"）
