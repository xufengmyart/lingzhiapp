```python

# 修改为真实的Bot链接：
```


## 🎨 自定义二维码样式

### 修改颜色

编辑 `scripts/generate_qrcode.py`，找到以下行：

```python
# 修改二维码颜色
img = qr.make_image(fill_color="black", back_color="white")

# 改为其他颜色：
img = qr.make_image(fill_color="#667eea", back_color="white")
```

### 添加Logo

在二维码中心添加品牌Logo：

```python
from PIL import Image

# 加载Logo
logo = Image.open("assets/logo.png")

# 调整Logo大小
logo_size = img.size[0] // 5
logo = logo.resize((logo_size, logo_size))

# 计算Logo位置
logo_pos = ((img.size[0] - logo_size) // 2, (img.size[1] - logo_size) // 2)

# 粘贴Logo
img.paste(logo, logo_pos, mask=logo.split()[3] if logo.mode == 'RGBA' else None)
```

## 🔍 二维码最佳实践

### 1. 尺寸建议
- **最小尺寸**: 2cm × 2cm（印刷时）
- **推荐尺寸**: 5cm × 5cm（手机扫描）
- **最大尺寸**: 10cm × 10cm（户外广告）

### 2. 颜色建议
- **前景色**: 深色（黑色、深蓝、深紫）
- **背景色**: 浅色（白色、浅灰）
- **对比度**: 至少4.5:1（确保可扫描）

### 3. 容错率
- **L级 (7%)**: 适合简洁环境
- **M级 (15%)**: 推荐（平衡）
- **Q级 (25%)**: 适合打印
- **H级 (30%)**: 高要求场景

### 4. 留白
- 二维码四周至少留4个模块的白边
- 确保扫描器能识别边界

## 🔧 故障排查

### 问题1：二维码无法扫描
**解决方案：**
- 检查对比度是否足够
- 确保二维码周围有足够留白
- 重新生成更高分辨率的二维码

### 问题2：链接跳转错误
**解决方案：**
- 验证Bot链接是否正确
- 确认智能体已发布并可见
- 检查链接是否过期

### 问题3：二维码模糊
**解决方案：**
- 使用矢量格式（SVG）生成
- 提高图片分辨率
- 确保打印质量
