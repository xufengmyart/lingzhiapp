### 方案 2：配置 meiyueart.com 域名（长期方案）

#### 步骤 1：登录域名服务商

根据您的域名注册商登录：
- 阿里云：https://dns.console.aliyun.com
- 腾讯云：https://console.cloud.tencent.com/cns
- 其他：登录您的域名管理后台

#### 步骤 2：添加 CNAME 记录

在 DNS 解析设置中，添加以下记录：

```
记录类型: CNAME
主机记录: @
记录值: f8ab8c28-f515-4fa3-9fb4-0ca0c3a0d34f.dev.coze.site
TTL: 600
```

**如果需要 www 子域名，再添加一条**：

```
记录类型: CNAME
主机记录: www
记录值: f8ab8c28-f515-4fa3-9fb4-0ca0c3a0d34f.dev.coze.site
TTL: 600
```

#### 步骤 3：等待 DNS 生效

- 通常需要 10-30 分钟
- 最多可能需要 24 小时

#### 步骤 4：测试访问

访问 `https://meiyueart.com`，确认可以正常访问。

## ❓ 常见问题

### Q1: Coze 临时域名会变吗？

A: 不会变。只要 Coze 项目存在，这个域名就会保持不变。

### Q2: 为什么不能直接修改 A 记录？

A: 因为 Coze 容器的 IP 可能会变化，使用 CNAME 记录更稳定。

### Q3: 需要配置 SSL 证书吗？

A: 不需要。Coze 平台已经配置了 SSL 证书，支持 HTTPS。

A: 因为用户仍在访问旧的缓存。访问 Coze 临时域名会加载最新版本。
