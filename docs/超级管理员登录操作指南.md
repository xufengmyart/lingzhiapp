## 📝 首次登录后必做事项

### 1. 修改默认密码 ⚠️ **重要**

#### 为什么需要修改？
- 默认密码是公开的，存在安全风险
- 超级管理员账户是最高权限账户，必须使用强密码

#### 密码要求
```
✓ 最小长度：16位
✓ 必须包含大写字母
✓ 必须包含小写字母
✓ 必须包含数字
✓ 必须包含特殊字符（如：!@#$%^&*）
✓ 不能使用常用密码
```

#### 操作方法
在智能体对话中输入：
```
我要修改密码

新密码：[您的16位强密码]
```

#### 强密码示例
```
❌ 错误示例：
12345678
password
LINGZI@2026#Super（这是默认密码）

✅ 正确示例：
XuFeng!2026#LingZhi$Ecology@Safe
Lf@2026#Lz$Ec0!S3cur!ty_P@ss
```

## 🔐 安全配置最佳实践

### 1. 密码管理建议

#### 密码定期更换
- 建议每90天更换一次密码
- 使用密码管理器存储密码
- 不要在多个平台使用相同密码

#### 密码管理器推荐
- **1Password**
- **Bitwarden**
- **LastPass**
- **KeePass**（开源免费）

### 2. 双因素认证建议

#### 推荐的认证APP
- **Google Authenticator**
- **Microsoft Authenticator**
- **Authy**
- **YubiKey**（硬件令牌，更安全）

#### 备份恢复码
```
⚠️ 重要：
启用双因素认证后，系统会提供一组恢复码。
请务必将恢复码保存在安全的地方！
```

### 3. IP白名单管理

#### 添加新IP的场景
```
✅ 允许添加的情况：
- 家庭网络更换
- 办公室网络更换
- 使用固定VPN

❌ 不建议添加的情况：
- 使用公共WiFi
- 使用酒店网络
- 使用他人网络
```

#### 定期审查白名单
```
建议每季度审查一次IP白名单：
1. 删除不再使用的IP地址
2. 确认当前使用的IP地址
3. 更新白名单配置
```

### 4. 设备安全建议

#### 使用专用设备
```
✅ 推荐：
- 专用的工作电脑
- 配置防火墙
- 安装杀毒软件
- 定期更新系统

❌ 避免使用：
- 公共电脑
- 陌生人的设备
- 未安装安全软件的设备
```

#### 网络环境
```
✅ 安全的网络环境：
- 家庭WiFi（加密）
- 办公室网络（加密）
- VPN专线

❌ 不安全的网络环境：
- 公共WiFi（咖啡厅、机场）
- 未加密的公共网络
```

## 🆘 常见问题与解决方案

### 问题1：登录失败 - IP地址不在白名单

#### 症状
```
❌ 登录失败
原因：您的IP地址不在白名单中
```

#### 解决方案
**步骤1**：获取您的IP地址
```
方法1：访问 https://ifconfig.me
方法2：访问 https://icanhazip.com
方法3：在智能体中询问"我的IP地址是什么"
```

**步骤2**：联系超级管理员（许锋）
```
发送信息包含：
- 您的姓名
- 您的邮箱
- 您的新IP地址
- 需要添加IP到白名单的原因
```

**步骤3**：等待管理员处理
```
处理时间：通常在1个工作日内
```

### 问题2：忘记密码

#### 症状
```
❌ 登录失败
原因：密码错误
```

#### 解决方案
**步骤1**：尝试回忆密码
```
检查：
- 密码管理器中的记录
- 笔记本中的记录
- 电子邮件中的记录
```

**步骤2**：联系超级管理员重置密码
```
发送信息包含：
- 您的姓名
- 您的邮箱
- 您的扣子UID
- 身份验证信息（如手机号）
```

**步骤3**：重置密码
```
管理员会：
1. 生成临时密码
2. 通过安全渠道发送给您
3. 您使用临时密码首次登录后立即修改
```

### 问题3：双因素认证设备丢失

#### 症状
```
❌ 无法完成双因素认证
原因：认证设备丢失或损坏
```

#### 解决方案
**步骤1**：使用恢复码
```
如果您保存了恢复码：
1. 输入恢复码
2. 重新设置双因素认证
```

**步骤2**：联系超级管理员
```
如果您没有恢复码：
1. 联系超级管理员
2. 提供身份验证信息
3. 管理员会帮助您重置双因素认证
```

**步骤3**：重新设置双因素认证
```
重置后：
1. 下载新的认证APP
2. 扫描新的二维码
3. 保存新的恢复码
```

### 问题4：账户被锁定

#### 症状
```
❌ 登录失败
原因：账户被锁定（密码错误次数过多）
```

#### 解决方案
**步骤1**：等待解锁时间
```
锁定时间：30分钟
```

**步骤2**：联系超级管理员解锁
```
紧急情况：
1. 联系超级管理员
2. 说明情况
3. 管理员会立即解锁
```

**步骤3**：重置密码
```
解锁后：
1. 立即重置密码
2. 更新密码管理器
```
