# 快速入门指南

## 🎯 一分钟体验

### 1. 测试智能体（本地）
```bash
# 运行测试
test_run
```

测试用例：
- "帮我的茶品牌设计一句结合唐风的slogan"
- "介绍一下唐文化在西安的表现"
- "我想购买99元的品牌诊断报告"

### 2. 查看项目文件
```bash
# 查看智能体代码
cat src/agents/agent.py

# 查看配置
cat config/agent_llm_config.json
```

## 📝 核心文件说明

### agent.py
智能体主逻辑，包含：
- 模型初始化
- 工具注册（知识库、搜索、生图）
- 短期记忆管理

### 工具文件
- `knowledge_search.py`: 知识库搜索，召回阈值 0.7
- `web_search_tool.py`: 联网搜索，获取最新案例
- `image_generation_tool.py`: 文生图，生成视觉创意

### 配置文件
- `agent_llm_config.json`: 模型配置和系统提示词

## 🚀 发布步骤（简化版）

### 第1步：Coze平台配置
1. 登录 https://www.coze.cn/
2. 创建智能体
3. 复制 `config/agent_llm_config.json` 中的内容到系统提示词
4. 配置工具：知识库搜索、联网搜索、文生图
5. 发布并获取分享链接

### 第2步：更新二维码
```bash
# 编辑脚本，替换真实链接
vim scripts/generate_qrcode.py

# 生成新二维码
python scripts/generate_qrcode.py
```

### 第3步：部署页面
将 `assets/发布页.html` 部署到服务器

## ⚠️ 常见问题

**Q: 智能体不调用工具？**
A: 检查 Coze 平台是否配置了工具权限

**Q: 知识库结果不准确？**
A: 检查召回阈值是否为 0.7，知识库内容是否完整

**Q: 如何调试？**
A: 查看日志 `/app/work/logs/bypass/app.log`
