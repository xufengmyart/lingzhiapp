## 🎯 技术架构

### 前端技术栈

- **框架**：React 18.3.1 + TypeScript 5.4.5
- **构建工具**：Vite 5.4.21
- **样式**：Tailwind CSS 3.4.3
- **路由**：React Router 6.22.3
- **状态管理**：Context API
- **图标**：Lucide React

### 后端技术栈

- **框架**：Python Flask
- **数据库**：PostgreSQL
- **认证**：JWT

### 部署架构

- **服务器**：阿里云 ECS
- **Web 服务器**：Nginx
- **CI/CD**：自动化部署系统
- **Git**：GitHub

## 🛠️ 开发指南

### 项目结构

```
web-app/
├── src/
│   ├── components/     # 公共组件
│   ├── contexts/       # 上下文
│   ├── pages/          # 页面组件
│   ├── services/       # API 服务
│   ├── utils/          # 工具函数
│   └── index.css       # 全局样式
├── public/             # 静态资源
└── package.json        # 依赖配置
```

### 可用脚本

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
