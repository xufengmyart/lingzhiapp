## ❓ 常见问题

### Q1：解压失败怎么办？

**解决方案**：
1. 确认下载的文件完整（验证校验和）
2. 检查是否有足够的磁盘空间
3. 使用不同的解压工具（7-Zip、WinRAR、tar）

### Q2：依赖安装失败怎么办？

**解决方案**：
```bash
# 升级 pip
python -m pip install --upgrade pip

# 单独安装失败的包
pip install 包名

# 如果是网络问题，使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q3：数据库初始化失败怎么办？

**解决方案**：
1. 检查 Python 版本是否 >= 3.10
2. 确认所有依赖已安装
3. 删除 `src/auth/auth.db` 文件后重新初始化
4. 查看错误日志

### Q4：服务无法启动怎么办？

**解决方案**：
1. 检查端口 8000 是否被占用：
   ```bash
   netstat -tuln | grep :8000
   ```
2. 如果被占用，更换端口：
   ```bash
   python -m uvicorn src.main:app --host 0.0.0.0 --port 8001 --reload
   ```
3. 查看错误日志

### Q5：收款码无法显示怎么办？

**解决方案**：
1. 确认文件名完全匹配（区分大小写）
2. 确认文件格式为 JPG
3. 检查文件是否损坏
4. 验证文件路径是否正确
