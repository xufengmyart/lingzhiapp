## 🚀 3分钟快速安装

### Windows 用户

```powershell
# 1. 解压文件（使用 7-Zip 或右键解压）
# 2. 进入 web-app 目录
cd web-app

# 3. 安装依赖
npm install

# 4. 构建项目
npm run build

# 5. 启动服务器
start-production.bat

# 6. 打开浏览器访问
# http://localhost:3000
```

### Linux / macOS 用户

```bash
# 1. 解压文件
tar -xzf lingzhi-ecosystem-webapp-v7.2.tar.gz

# 2. 进入 web-app 目录
cd web-app

# 3. 安装依赖
npm install

# 4. 构建项目
npm run build

# 5. 启动服务器
./start-production.sh

# 6. 打开浏览器访问
# http://localhost:3000
```
