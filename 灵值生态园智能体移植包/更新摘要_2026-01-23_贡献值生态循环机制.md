## 🚀 贡献值生态循环机制

### 核心理念

**贡献值不仅仅是收入，更是参与生态的"通行证"和"投资工具"。**

### 生态循环

```
用户 → 完成任务 → 获得贡献值（收入）
         ↓
    参与项目 → 消耗贡献值（参与门槛，5%-20%）
         ↓
    项目成功 → 获得奖励（返还消耗 + 高额奖励，ROI 600%-12400%）
         ↓
    重复循环 → 生态繁荣
```

### 项目参与规则

**1. 项目估值标准**
- 轻量任务：50-200贡献值
- 中等项目：200-500贡献值
- 重点项目：500-1000贡献值
- 核心项目：1000-5000贡献值

**2. 参与门槛（消耗贡献值）**
- 轻量任务：消耗20%（10-40贡献值）
- 中等项目：消耗15%（30-75贡献值）
- 重点项目：消耗10%（50-100贡献值）
- 核心项目：消耗5%（50-250贡献值）

**3. 奖励机制**
- 消耗返还：合格成果全额返还
- 基础奖励：项目估值 × 贡献度 × 奖励系数（1.5-10.0）
- 质量奖励：A级×1.5，S级×2.0
- 创新奖励：+10%-30%
- 总收益 = 返还消耗 + 基础奖励 × 质量倍数 × 创新倍数

**4. 奖励计算示例**

示例1：轻量任务
```
项目估值：100贡献值
消耗贡献值：20贡献值（20%）
完成任务（A级质量）：
- 基础奖励：100 × 2.0 = 200贡献值
- 质量奖励：200 × 1.5 = 300贡献值
- 返还消耗：20贡献值
- 总获得：320贡献值
- 净收益：300贡献值
- ROI：1500%
```

示例2：重点项目
```
项目估值：800贡献值
消耗贡献值：80贡献值（10%）
完成任务（S级质量+创新）：
- 基础奖励：800 × 5.0 = 4000贡献值
- 质量奖励：4000 × 2.0 = 8000贡献值
- 创新奖励：8000 × 1.2 = 9600贡献值
- 返还消耗：80贡献值
- 总获得：9680贡献值
- 净收益：9600贡献值
- ROI：12000%
```
