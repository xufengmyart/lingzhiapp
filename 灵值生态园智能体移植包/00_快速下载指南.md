# 快速下载指南

> **文件大小**：91KB（压缩后）
> **预计下载时间**：<1秒（100Mbps网络）

---

## 📥 下载方法

### 方法1：下载压缩包（推荐）

**步骤**：

1. **在本地打开终端**
   ```bash
   # Mac/Linux
   open Terminal

   # Windows
   打开 PowerShell 或 Git Bash
   ```

2. **下载压缩包**
   ```bash
   # 替换以下信息：
   # user@remote_host -> 您的远程服务器地址和用户名

   scp user@remote_host:/workspace/projects/灵值生态园智能体移植包.tar.gz ~/Desktop/
   ```

3. **解压文件**
   ```bash
   # 进入桌面目录
   cd ~/Desktop

   # 解压文件
   tar -xzf 灵值生态园智能体移植包.tar.gz
   ```

4. **验证文件**
   ```bash
   # 进入移植包目录
   cd 灵值生态园智能体移植包

   # 运行验证脚本
   python 05_移植验证脚本.py
   ```

### 方法2：复制整个目录

```bash
# 复制整个移植包到本地
scp -r user@remote_host:/workspace/projects/灵值生态园智能体移植包 ~/Desktop/
```

### 方法3：使用 rsync（更快）

```bash
# 使用 rsync 下载（支持断点续传）
rsync -avz --progress user@remote_host:/workspace/projects/灵值生态园智能体移植包 ~/Desktop/
```
