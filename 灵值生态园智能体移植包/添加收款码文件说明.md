# 如何添加收款码文件

## 📌 重要提示

在发布智能体之前，**必须**添加两个收款码文件到 `assets/` 目录：

1. **个人微信收款码.jpg**
2. **个人支付宝收款码.jpg**

## 🚀 快速操作指南

### 方法1：使用命令行复制文件

```bash
# 进入项目目录
cd 灵值生态园智能体移植包

# 复制微信收款码
cp /path/to/你的微信收款码.jpg assets/个人微信收款码.jpg

# 复制支付宝收款码
cp /path/to/你的支付宝收款码.jpg assets/个人支付宝收款码.jpg

# 验证文件是否已添加
ls -lh assets/*收款码*.jpg
```

### 方法2：手动拖拽（推荐）

1. 打开 `灵值生态园智能体移植包/assets/` 文件夹
2. 将你的微信收款码图片拖入该文件夹
3. 将你的支付宝收款码图片拖入该文件夹
4. 确认文件名：
   - 微信收款码重命名为：`个人微信收款码.jpg`
   - 支付宝收款码重命名为：`个人支付宝收款码.jpg`

## ✅ 验证步骤

```bash
cd 灵值生态园智能体移植包
```

## ⚠️ 常见问题

### Q1：文件名不匹配怎么办？

**解决方案**：
```bash
# 重命名文件
mv assets/当前文件名.jpg assets/个人微信收款码.jpg
mv assets/当前文件名.jpg assets/个人支付宝收款码.jpg
```

### Q2：图片无法显示？

**可能原因**：
1. 文件名不匹配
2. 文件不在正确的目录
3. 文件格式不是 JPG

**解决方案**：
1. 检查文件名是否完全一致（区分大小写）
2. 确认文件在 `assets/` 目录下
3. 将图片转换为 JPG 格式

### Q3：二维码无法扫描？

**解决方案**：
1. 确认收款码图片清晰
2. 检查是否有水印或遮挡
3. 使用手机扫描测试
