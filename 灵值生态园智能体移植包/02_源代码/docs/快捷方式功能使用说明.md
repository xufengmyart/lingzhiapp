# 快捷方式功能使用说明

## 📱 快速开始

### 手机用户（最常用）

#### iPhone/iPad 用户
```
1. 在Safari浏览器中打开智能体
2. 点击分享按钮（底部的方形图标）
3. 选择"添加到主屏幕"
4. 点击"添加"

完成！主屏幕会出现智能体图标，一键即可进入。
```

#### 安卓手机用户
```
1. 在Chrome浏览器中打开智能体
2. 点击右上角菜单（三个点）
3. 选择"添加到主屏幕"
4. 点击"添加"

完成！主屏幕会出现智能体图标，一键即可进入。
```

---

### 电脑用户

#### Windows 用户
```
方法1：创建桌面快捷方式
1. 桌面右键 → 新建 → 快捷方式
2. 粘贴智能体链接
3. 命名为"灵值生态园"
4. 完成

方法2：固定到任务栏（推荐）
1. 在Chrome或Edge中打开智能体
2. 点击地址栏右侧的锁形图标
3. 选择"固定到任务栏"

完成！任务栏会出现智能体图标。
```

#### Mac 用户
```
方法：添加到Dock栏
1. 在Safari中打开智能体
2. 点击地址栏左侧的URL图标
3. 拖拽到Dock栏右侧

完成！Dock栏会出现智能体图标。
```

## 💡 常见问题

### Q：每次都要扫码吗？
**A：** 不需要！创建快捷方式后，直接点击图标即可进入。

### Q：哪个方法最简单？
**A：**
- 手机：添加到主屏幕（1分钟搞定）
- 电脑：固定到任务栏/Dock栏（30秒搞定）

### Q：快捷方式会占用手机空间吗？
**A：** 不会，只是指向链接的快捷方式，几乎不占空间。

### Q：可以删除快捷方式吗？
**A：** 可以，长按图标删除即可，不影响智能体使用。

### Q：我找不到"添加到主屏幕"选项？
**A：**
- iPhone：确保用Safari浏览器打开
- 安卓：确保用Chrome浏览器打开
- 也可以尝试刷新页面
