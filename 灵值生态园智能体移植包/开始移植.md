## 🚀 快速移植（5分钟）

### Step 1：阅读主文档
```
打开：README_移植指南.md
```

### Step 2：配置智能体
```
1. 复制：config/02_系统提示词.md → 平台提示词
2. 复制：config/03_模型配置.json → 平台配置
```

### Step 3：上传知识库
```
上传：assets/灵值生态一体化服务总纲.md
设置：最高优先级，召回阈值0.7
```

### Step 4：创建工作流
```
参考：docs/Coze平台工作流配置指南.md
创建：核心路由工作流
```

### Step 5：测试发布
```
运行：scripts/测试脚本.py
发布：智能体
```
