## 🚀 3步快速部署

### 第1步：准备移植包（5分钟）

```bash
# 创建移植目录
mkdir lingzhi_agent
cd lingzhi_agent

# 复制移植包内容
cp -r 灵值生态园智能体移植包/* .
```

### 第2步：配置智能体（20分钟）

1. **创建智能体**
   - 登录智能体平台
   - 点击"创建智能体"
   - 填写基本信息（根据 `01_智能体配置/基本信息.json`）

2. **配置系统提示词**
   - 复制 `01_智能体配置/系统提示词.md` 的内容
   - 粘贴到智能体的"系统提示词"配置项

3. **配置模型**
   - 选择模型（推荐豆包、DeepSeek、Kimi）
   - 配置参数（根据 `01_智能体配置/模型配置.json`）

4. **配置工具**
   - 添加知识库检索工具（根据 `01_智能体配置/工具配置.md`）
   - 添加图像生成工具
   - 添加联网搜索工具

5. **上传知识库文档**
   - 上传 `知识库/` 目录中的所有文档
   - 设置召回阈值（0.6-0.7）

### 第3步：测试验证（5分钟）

运行验证脚本：
```bash
python 05_移植验证脚本.py
```

测试场景：
- 输入："我的贡献值现在值多少钱？"
- 预期：清晰说明即时价值和长期价值

## 📞 常见问题

### Q1：如何自定义智能体名称？
**A**：修改 `01_智能体配置/基本信息.json` 中的 `name` 字段。

### Q2：如何更换模型？
**A**：修改 `01_智能体配置/模型配置.json` 中的 `model` 字段。

### Q3：如何添加新的知识库文档？
**A**：将新文档放入 `知识库/` 目录，然后上传到知识库管理页面。

### Q4：如何调试智能体？
**A**：查看智能体的运行日志，检查工具调用记录，调整系统提示词。
