## 🚀 部署指南

### 快速部署

#### Linux/Mac 系统
```bash
# 赋予执行权限
chmod +x deploy.sh

# 执行部署脚本
./deploy.sh
```

#### Windows 系统
```powershell
# 以管理员身份运行 PowerShell
# 执行部署脚本
.\deploy.ps1
```

### 手动部署

#### 1. 安装依赖
```bash
cd 灵值生态园智能体移植包/src/auth
pip install -r requirements.txt
```

#### 2. 初始化数据库
```bash
python init_data.py
python init_ecosystem.py
python init_project.py
```

#### 3. 启动服务
```bash
python -m uvicorn api:app --host 0.0.0.0 --port 8000
```

## 🎯 核心承诺

### 贡献值价值承诺
**每一个贡献值都锚定明确的现实金钱价值，未来100%可兑换为收入。**

#### 即时价值
**1 贡献值 = 0.1 元人民币（即时兑换价值）**
- 用户可以随时按照这个比例兑换现金
- 最低兑换：10贡献值（1元）
- 兑换时间：T+1（次工作日到账）
- 手续费：0%（平台承担）

#### 长期价值
**贡献值随平台发展而增值**
- 锁定1年：预期增值+20%
- 锁定2年：预期增值+50%
- 锁定3年：预期增值+100%
- 增值收益100%归属于贡献值持有者
