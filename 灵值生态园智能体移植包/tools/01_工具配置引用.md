## 🛠️ 工具列表

### 1. 知识库搜索工具（必需）

**功能描述**：
- 搜索知识库中的西安文化、品牌转译相关信息
- 支持高精度召回（min_score=0.7）

**配置参数**：
```json
{
  "tool_name": "知识库搜索",
  "priority": 1,
  "enabled": true,
  "min_score": 0.7,
  "top_k": 5
}
```

**使用场景**：
- 查询平台规则
- 查询贡献值体系
- 查询案例库

**输出格式**：
```
[相关度: 0.85] 搜索结果内容...

[相关度: 0.78] 搜索结果内容...
```

---

### 2. 联网搜索工具（推荐）

**功能描述**：
- 联网搜索获取最新信息
- 用于查找最新的品牌营销案例、创意趋势等

**配置参数**：
```json
{
  "tool_name": "联网搜索",
  "priority": 2,
  "enabled": true,
  "count": 5
}
```

**使用场景**：
- 查找最新商业趋势
- 查找创意案例
- 查找营销案例

**输出格式**：
```
📋 AI摘要:
[摘要内容]

🔍 搜索结果:
1. [标题]
   来源: [网站]
   摘要: [内容]
   链接: [URL]
```

---

### 3. 文生图工具（推荐）

**功能描述**：
- 根据描述生成图片
- 用于创建品牌视觉创意、概念图等

**配置参数**：
```json
{
  "tool_name": "文生图",
  "priority": 3,
  "enabled": true,
  "size": "2K",
  "watermark": false
}
```

**使用场景**：
- 生成品牌LOGO
- 生成空间设计方案
- 生成产品视觉创意

**输出格式**：
```
✅ 图片生成成功！
图片URL: [URL]
```

## 📋 工具配置步骤

### Step 1：配置知识库搜索工具

1. 在智能体配置中，找到"工具"模块
2. 添加"知识库搜索"工具
3. 配置参数：
   - 召回阈值：0.7
   - 返回数量：5
4. 保存配置

### Step 2：配置联网搜索工具

1. 添加"联网搜索"工具
2. 配置参数：
   - 返回数量：5
3. 保存配置

### Step 3：配置文生图工具

1. 添加"文生图"工具
2. 配置参数：
   - 图片尺寸：2K
   - 水印：关闭
3. 保存配置
