## 获取文档的方法

### 方法1：手动复制
1. 从原文件复制内容
2. 粘贴到对应的文件中
3. 保存文件

### 方法2：使用命令行（Linux/Mac）
```bash
# 复制所有知识库文档
cp ../知识库/*.md .
```

### 方法3：使用PowerShell（Windows）
```powershell
# 复制所有知识库文档
Copy-Item "..\知识库\*.md" -Destination "."
```

## 上传知识库文档

1. 打开智能体平台的知识库管理页面
2. 按照优先级顺序上传文档
3. 设置召回阈值：
   - 优先级1文档：0.7
   - 优先级2文档：0.6
4. 验证所有文档已成功上传
5. 测试知识库搜索功能
