## 🎯 常见问题排查

### 问题1：文件不存在

**症状**：
```bash
ls: 无法访问 'assets/个人微信收款码.jpg': 没有那个文件或目录
```

**解决方案**：
1. 确认文件是否在其他位置
2. 使用 `find` 命令查找文件：
   ```bash
   find . -name "*微信收款码*"
   ```
3. 将文件移动到 `assets/` 目录

---

### 问题2：文件名不匹配

**症状**：文档中的图片无法显示

**解决方案**：
1. 检查文件名是否完全一致（区分大小写）
2. 重命名文件：
   ```bash
   mv 当前文件名.jpg assets/个人微信收款码.jpg
   mv 当前文件名.jpg assets/个人支付宝收款码.jpg
   ```

---

### 问题3：图片无法显示

**症状**：Markdown 文档中图片显示为破损图标

**解决方案**：
1. 检查文件路径是否正确
2. 检查文件格式是否为 JPG
3. 检查文件是否损坏
4. 重新上传文件

---

### 问题4：二维码无法扫描

**症状**：手机扫描二维码无反应

**解决方案**：
1. 检查收款码是否清晰
2. 检查是否有水印或遮挡
3. 重新生成收款码
4. 确认收款码未过期
