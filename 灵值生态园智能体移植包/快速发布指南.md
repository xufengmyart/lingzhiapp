## 🔄 故障排除

### 服务无法启动
```bash
# 查看服务日志
tail -f /tmp/api.log  # Linux/Mac
type C:\temp\api.log  # Windows

# 检查端口占用
lsof -i :8000  # Linux/Mac
netstat -ano | findstr :8000  # Windows
```

### 数据库问题
```bash
# 检查数据库文件
ls -lh src/auth/auth.db  # Linux/Mac
dir src\auth\auth.db  # Windows

# 重新初始化数据库
cd src/auth
python init_data.py
python init_ecosystem.py
python init_project.py
```

### 依赖问题
```bash
# 重新安装依赖
pip install -r requirements.txt --force-reinstall
```
