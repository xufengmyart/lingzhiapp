## 🚀 快速开始

### 第一步：上传知识库文档

1. 登录目标平台（如Coze）
2. 进入"知识库"模块
3. 按以下顺序上传文档：
   - `assets/01_灵值生态一体化服务总纲.md`（设为最高优先级）
   - `assets/02_西安文化基因库.md`
   - `assets/03_转译商业案例库.md`
4. 设置召回阈值：`min_score = 0.7`

### 第二步：创建工作流

1. 进入"工作流"模块
2. 按照以下顺序创建工作流：
   - `01_核心路由_用户意图分析.json`
   - `02_双螺旋诊断_定制化建议.json`
   - `03_贡献值诠释_数值与文化.json`
3. 发布所有工作流

### 第三步：创建智能体

1. 进入"智能体"模块
2. 创建新智能体
3. 配置基本信息（使用 `config/01_智能体基本信息.json`）
4. 配置系统提示词（使用 `config/02_系统提示词.md`）
5. 配置模型参数（使用 `config/03_模型配置.json`）
6. 添加工具（使用 `tools/` 目录下的配置）
7. 关联工作流（关联第二步创建的工作流）

### 第四步：测试验证

1. 运行 `scripts/02_测试脚本.py`
2. 根据测试结果调整配置
3. 使用 `docs/03_功能验证清单.md` 逐项验证

### 第五步：发布上线

1. 参考 `docs/02_发布步骤详解.md`
2. 生成推广材料（使用 `assets_promo/` 目录）
3. 发布智能体

## 🔧 常见问题

### Q1：如何调整模型参数？
A：修改 `config/03_模型配置.json`，建议：
- 温度：0.8（平衡创意与准确）
- Top_p：0.9（控制多样性）
- 最大Token：4096（确保完整回复）

### Q2：如何调整知识库召回阈值？
A：在知识库设置中，将 `min_score` 设置为 0.7（高精度）

### Q3：工作流调用失败怎么办？
A：
1. 检查工作流是否已发布
2. 检查输入输出变量是否匹配
3. 查看工作流日志定位问题

### Q4：如何优化回复质量？
A：
1. 调整系统提示词中的心法
2. 优化知识库内容
3. 调整模型温度参数
