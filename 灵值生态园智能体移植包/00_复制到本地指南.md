## 方法1：使用命令行（推荐）

### Linux/Mac 用户

#### 1.1 使用 scp 命令（推荐）

```bash
# 复制整个移植包到本地
scp -r user@remote_host:/workspace/projects/灵值生态园智能体移植包 ~/Desktop/

# 示例：假设远程主机为 192.168.1.100，用户名为 root
scp -r root@192.168.1.100:/workspace/projects/灵值生态园智能体移植包 ~/Desktop/
```

**参数说明**：
- `-r`：递归复制整个目录
- `user@remote_host`：远程主机地址和用户名
- `~/Desktop/`：本地目标目录

#### 1.2 使用 rsync 命令（更高效）

```bash
# 复制整个移植包到本地（推荐）
rsync -avz user@remote_host:/workspace/projects/灵值生态园智能体移植包 ~/Desktop/

# 示例：假设远程主机为 192.168.1.100，用户名为 root
rsync -avz root@192.168.1.100:/workspace/projects/灵值生态园智能体移植包 ~/Desktop/
```

**参数说明**：
- `-a`：归档模式，保留文件属性
- `-v`：显示详细输出
- `-z`：压缩传输，提高传输速度

## 方法2：使用图形界面

### 2.1 使用 FileZilla（跨平台）

**步骤**：

1. **下载 FileZilla**
   - 官网：https://filezilla-project.org/
   - 下载并安装

2. **配置连接**
   - 打开 FileZilla
   - 点击"文件" -> "站点管理器"
   - 新建站点
   - 填写连接信息：
     - 主机：远程主机地址（如 192.168.1.100）
     - 协议：SFTP
     - 用户名：远程用户名
     - 密码：远程密码
   - 点击"连接"

3. **下载移植包**
   - 左侧：本地文件系统，选择目标目录
   - 右侧：远程文件系统，导航到 `/workspace/projects/灵值生态园智能体移植包`
   - 右键点击"灵值生态园智能体移植包"文件夹
   - 选择"下载"

### 2.2 使用 WinSCP（Windows）

**步骤**：

1. **下载 WinSCP**
   - 官网：https://winscp.net/
   - 下载并安装

2. **配置连接**
   - 打开 WinSCP
   - 新建站点
   - 填写连接信息：
     - 主机名：远程主机地址（如 192.168.1.100）
     - 文件协议：SFTP
     - 用户名：远程用户名
     - 密码：远程密码
   - 点击"登录"

3. **下载移植包**
   - 左侧：本地文件系统，选择目标目录
   - 右侧：远程文件系统，导航到 `/workspace/projects/灵值生态园智能体移植包`
   - 右键点击"灵值生态园智能体移植包"文件夹
   - 选择"下载"

## 方法3：使用云存储

### 3.1 打包上传到云存储

#### 在远程服务器上打包

```bash
# 进入项目目录
cd /workspace/projects

# 打包为 zip 文件
zip -r 灵值生态园智能体移植包.zip 灵值生态园智能体移植包/

# 打包为 tar.gz 文件（推荐，更小）
tar -czf 灵值生态园智能体移植包.tar.gz 灵值生态园智能体移植包/
```

#### 上传到云存储

```bash
# 上传到阿里云 OSS（需要安装 ossutil）
ossutil cp 灵值生态园智能体移植包.tar.gz oss://your-bucket/

# 上传到腾讯云 COS（需要安装 coscli）
coscli cp 灵值生态园智能体移植包.tar.gz cos://your-bucket/

# 上传到百度云 BOS（需要安装 bcecmd）
bcecmd cp 灵值生态园智能体移植包.tar.gz bos://your-bucket/
```

#### 在本地下载

1. 登录云存储控制台
2. 找到上传的文件
3. 点击下载

### 3.2 使用网盘

#### 上传到百度网盘

1. 在远程服务器上安装 BaiduPCS-Go
2. 上传文件
```bash
BaiduPCS-Go upload 灵值生态园智能体移植包.tar.gz /灵值生态园智能体移植包.tar.gz
```

#### 上传到阿里云盘

1. 在远程服务器上安装 aliyundrive-webdav
2. 挂载为本地目录
3. 复制文件

## 常见问题

### Q1：复制速度很慢怎么办？

**A**：使用压缩文件 + rsync
```bash
# 在远程服务器上打包
tar -czf 灵值生态园智能体移植包.tar.gz 灵值生态园智能体移植包/

# 使用 rsync 传输
rsync -avz --progress user@remote_host:/workspace/projects/灵值生态园智能体移植包.tar.gz ~/Desktop/
```

### Q2：复制过程中断怎么办？

**A**：使用 rsync 的断点续传功能
```bash
# rsync 会自动断点续传
rsync -avz --partial user@remote_host:/workspace/projects/灵值生态园智能体移植包 ~/Desktop/
```

### Q3：如何验证文件是否完整？

**A**：运行验证脚本
```bash
cd 灵值生态园智能体移植包
python 05_移植验证脚本.py
```

### Q4：复制后文件损坏怎么办？

**A**：重新复制或使用压缩文件
```bash
# 使用压缩文件（更可靠）
scp user@remote_host:/workspace/projects/灵值生态园智能体移植包.tar.gz ~/Desktop/
```
